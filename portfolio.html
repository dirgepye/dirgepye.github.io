War!

HTML • CSS • Javascript (jQuery)

A simple version of the classic card game "War!". The game keeps track of how many cards you have reheading in your deck. The cards are individual objects that are reflected on the front-end using HTML, CSS and repeating basic images for the suits. By creating a new card each time, rather than having 52 seperate images to represent each card in the deck, the file size is considerably smaller than what one would expect.
